## ğŸ“¦ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

-   Node.js 18+
-   npm or pnpm

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/smg-motors.git
    cd smg-motors/app-polish-and-ui
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Visit `http://localhost:3000` to see the app in action.
